## Usage

This poster template is publicly available under the Creative Commons
Attribution 4.0 International License (CC BY 4.0) and can be freely used,
modified, and shared.

It is designed for portrait-format posters and is compatible with
academic conferences that support vertical poster layouts, such as
NeurIPS, AAAI, ACL, IROS, ICRA, etc.
